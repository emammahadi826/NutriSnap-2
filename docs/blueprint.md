# **App Name**: NutriSnap

## Core Features:

- AI Food Recognition: Use the Gemini API as a tool to identify foods from user-uploaded images and estimate portion sizes.
- Nutrition Data Retrieval: Fetch calorie and macro information from a pre-existing, limited internal dataset.
- Meal Logging: Allow users to log meals with identified foods and adjusted portion sizes.
- Dashboard: Display daily calorie and macro totals in a clear, simple dashboard.
- Offline Caching: Cache recent lookups for usage when offline.

## Style Guidelines:

- Primary color: Fresh Mint (#98FF98) to evoke health and naturalness.
- Background color: Light Beige (#F5F5DC) for a clean and appetizing feel.
- Accent color: Carrot Orange (#FFA07A) to highlight key actions and information.
- Body and headline font: 'PT Sans', a humanist sans-serif for a balance of modernity and warmth.
- Use flat, simple icons related to food groups and nutritional information.
- Clean and spacious layout to make it easy to scan information at a glance.
- Subtle transitions and animations to provide feedback when logging meals or updating the dashboard.
